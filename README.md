# Tagg

A tool to handle my mp3 collection

[![asciicast](https://asciinema.org/a/ANaoLiSWILkIL8IAfbpUfgsZM.png)](https://asciinema.org/a/ANaoLiSWILkIL8IAfbpUfgsZM)

## Requirements

- Python 3

## Development

Install requirements:

```sh
make setup
```

Run:

```sh
./buildout/bin/tagg
```

Build:

```sh
make build
```
